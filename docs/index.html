Install from source

  1. Download the latest zip and unzip.
  2. Open chrome://extensions and enable Developer mode.
  3. Click Load unpacked and select the unzipped extension/ folder.
  4. Click the extension icon → Settings. Pick a provider, paste your API key, save.
  5. Open any YouTube video and click the ✨ Summarize button in the action row.
